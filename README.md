# 王飞教授课题组网站

## 课题组简介
王飞教授课题组隶属于中国石油大学（华东），专注于气体水合物相平衡、低碳能源技术、新能源材料等前沿领域的研究与创新。

## 研究方向
1. **气体水合物相平衡** - 研究不同条件下气体水合物的形成、稳定性和分解机理
2. **低碳能源技术** - 开发二氧化碳捕集、利用与封存技术
3. **新能源材料** - 设计合成高性能能源材料
4. **计算模拟与人工智能** - 利用分子模拟、机器学习等方法预测材料性能

# 王飞教授课题组学术网站

## 项目简介

这是王飞教授课题组的学术团队网站，用于展示团队研究方向、科研成果、团队成员等信息。

## 网站结构

```
academic-website/
├── index.html          # 首页
├── team.html           # 团队成员页面（待创建）
├── research.html       # 研究方向页面（待创建）
├── publications.html   # 科研成果页面（待创建）
├── teaching.html       # 教学工作页面（待创建）
├── contact.html        # 联系我们页面（待创建）
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── main.js         # JavaScript文件
└── images/             # 图片资源目录（需创建）
    └── prof-wang.jpg   # 王飞教授照片
```

## 如何使用

### 1. 准备图片资源

在 `images/` 目录下放置以下图片：
- `prof-wang.jpg` - 王飞教授的照片

### 2. 直接打开网站

使用浏览器打开 `index.html` 即可查看网站。

### 3. 或使用本地服务器

如果要更好的预览效果，可以使用本地服务器：

```bash
# 如果安装了Python 3
python -m http.server 8080

# 或者使用Node.js的http-server
npx http-server -p 8080
```

然后访问 `http://localhost:8080`

## 与实验室管理系统集成

导航栏中的"内部管理"链接会打开实验室管理系统（`http://localhost:3001`），供团队成员内部使用。

## 自定义配置

### 修改颜色主题

在 `css/style.css` 文件开头的 `:root` 中修改CSS变量：

```css
:root {
    --primary-color: #1a73e8;      /* 主色调 */
    --secondary-color: #34a853;    /* 次要色 */
    --accent-color: #fbbc04;       /* 强调色 */
    /* ... 其他颜色 */
}
```

### 修改内容

直接编辑对应的HTML文件即可修改页面内容。

## 注意事项

1. 首页中的教授照片需要您提供并放置在 `images/prof-wang.jpg`
2. "内部管理"链接默认指向 `http://localhost:3001`，如果实验室管理系统部署在其他地址，请修改导航栏中的链接
3. 建议添加团队成员照片、研究成果图片等视觉元素以提升网站效果

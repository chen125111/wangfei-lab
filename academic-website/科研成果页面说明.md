# 科研成果页面设计说明

## 📚 功能特点

### 1. **统计概览**
- SCI论文总数：100+篇
- 中科院1区论文：41篇
- 中科院2区论文：41篇
- 最高影响因子：13.2

### 2. **智能筛选功能**
- **年份筛选**：2014-2025年任意年份
- **分区筛选**：中科院1区、2区、其他分区
- **关键词搜索**：支持论文标题、期刊名称搜索

### 3. **论文信息展示**
每篇论文包含：
- ✅ 论文标题（英文原文）
- ✅ 作者列表（通讯作者加粗标注）
- ✅ 期刊名称（斜体）
- ✅ 发表年份和卷期
- ✅ 影响因子（IF）
- ✅ 中科院分区标签
- ✅ DOI链接（直达出版社网页）

### 4. **分年度展示**
- 按年份倒序排列（最新在前）
- 每年论文数量统计
- 支持"显示更多"功能

### 5. **用户体验优化**
- 鼠标悬停高亮效果
- 平滑动画过渡
- 响应式设计（手机、平板、电脑）
- 空状态友好提示

## 🎨 设计亮点

### 视觉设计
1. **分区徽章**：
   - 中科院1区：绿色徽章
   - 中科院2区：蓝色徽章
   - 影响因子：橙色徽章

2. **交互反馈**：
   - 悬停效果：阴影加深 + 左移动画
   - 点击链接：颜色变化
   - 筛选结果：实时更新

3. **信息层次**：
   - 标题最突出
   - 期刊次之
   - 元数据再次

## 📝 使用说明

### 访问页面
在主页导航栏点击"科研成果"或直接访问：
```
academic-website/publications.html
```

### 筛选论文
1. 选择年份下拉菜单
2. 选择分区下拉菜单
3. 在搜索框输入关键词
4. 系统自动实时筛选

### 查看论文详情
点击任意论文的"DOI链接"按钮，将在新标签页打开出版社官方页面。

### 显示更多
点击"显示更多XX年论文"按钮，可以展开该年度的所有论文（功能占位）。
